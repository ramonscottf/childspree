# Child Spree — Davis Education Foundation

Digital nomination and parent intake system for Child Spree 2026.

Built by Scott Foster / Wicko Waypoint

## Stack
- Cloudflare Pages (frontend)
- Cloudflare D1 (database)
- Cloudflare R2 (video/file storage)
- Cloudflare Workers/Functions (API)

## Deploy
```bash
npm install
npm run dev      # local dev
npm run deploy   # deploy to Cloudflare Pages
```
